About



This project was made by me as my CO-301 Software Engineering innovative project for my MTE evaluation.


I got this idea because of a bad experience I had with the DMV in my country in the past. A website system like this would help us in modernizing the primitive and inefficient methods used in the DMV currently.


This website was built using HTML, CSS, Javascript and Bootstrap for the frontend, whereas Django and SQLite were used for the backend.
